# 云开发环境配置说明

## 问题描述

如果看到以下错误：
```
errCode: -501000 | errMsg: [100003] env not exists
```

说明云开发环境不存在或未正确配置。

## 重要说明

**即使使用 `wx.cloud.extend.AI` 直接调用 AI API，仍然需要正确的云开发环境配置。**

原因：
- `wx.cloud.extend.AI.createModel()` 可以创建模型实例
- 但是调用 `model.streamText()` 时，内部仍然需要访问云开发环境来获取访问令牌
- 因此，必须正确配置云开发环境

## 解决方案

### 方法一：在微信开发者工具中配置云开发环境（推荐）

1. **打开微信开发者工具**
   - 确保已登录微信开发者账号

2. **打开云开发控制台**
   - 点击工具栏上的 **"云开发"** 按钮
   - 或者在菜单栏选择 **"工具"** → **"云开发"**

3. **创建或选择环境**
   - 如果已有环境，选择正确的环境
   - 如果没有环境，点击 **"创建环境"** 创建新环境
   - 环境名称可以自定义（如：`xiaoshuo`）

4. **获取环境 ID**
   - 在云开发控制台中，找到 **"环境 ID"**
   - 复制环境 ID（格式类似：`xxxx-xxxx-xxxx`）

5. **配置环境 ID**
   - 打开 `miniprogram/app.js`
   - 将 `env: 'xiaoshuo-8g2t52vw6be64c9a'` 替换为你的环境 ID
   - 保存文件

6. **开通 AI 能力**
   - 在云开发控制台中，找到 **"AI"** 或 **"大模型"** 功能
   - 点击开通 AI 能力
   - 选择使用的模型（如：deepseek）

7. **重新编译**
   - 点击 **"编译"** 按钮重新编译小程序
   - 测试功能是否正常

### 方法二：检查现有环境配置

1. **检查环境 ID 是否正确**
   - 打开 `miniprogram/app.js`
   - 检查 `env: 'xiaoshuo-8g2t52vw6be64c9a'` 是否正确
   - 如果不正确，替换为正确的环境 ID

2. **检查环境是否开通 AI 能力**
   - 在云开发控制台中，查看环境是否开通了 AI 能力
   - 如果没有开通，点击开通

3. **检查环境状态**
   - 确保环境状态为 **"正常"**
   - 如果环境被禁用或删除，需要重新创建

### 方法三：使用动态环境（如果支持）

如果微信开发者工具支持动态环境选择，可以：

1. **不指定环境 ID**
   ```javascript
   wx.cloud.init({
     traceUser: true,
     // 不指定 env，使用默认环境
   });
   ```

2. **或者使用环境变量**
   ```javascript
   wx.cloud.init({
     env: wx.cloud.DYNAMIC_CURRENT_ENV, // 使用当前选择的环境
     traceUser: true,
   });
   ```

## 验证配置

配置完成后，重新编译小程序，查看控制台日志：

### 1. 应该看到：
```

✅ AI 能力可用
✅ AI 模型创建测试成功
✅ streamText 方法可用
```

### 2. 不应该看到：
```

❌ 云开发环境不存在
```

## 常见问题

### Q: 为什么需要云开发环境？

A: 即使使用 `wx.cloud.extend.AI` 直接调用 AI API，内部仍然需要访问云开发环境来获取访问令牌和认证信息。

### Q: 如何获取环境 ID？

A: 
1. 在微信开发者工具中，点击"云开发"按钮
2. 在云开发控制台中，找到"环境 ID"
3. 复制环境 ID

### Q: 环境 ID 格式是什么？

A: 环境 ID 格式通常为：`xxxx-xxxx-xxxx` 或 `xxxxxx`（字母和数字组合）

### Q: 如何开通 AI 能力？

A:
1. 在云开发控制台中，找到"AI"或"大模型"功能
2. 点击开通
3. 选择使用的模型
4. 确认开通

### Q: 环境不存在怎么办？

A:
1. 创建新的云开发环境
2. 获取环境 ID
3. 更新 `app.js` 中的环境 ID
4. 重新编译小程序

## 配置示例

### app.js 配置
```javascript
App({
  onLaunch() {
    if (!wx.cloud) {
      console.warn('请在 2.2.3 或以上基础库使用云能力');
    } else {
      try {
        // 初始化云开发，使用环境ID
        wx.cloud.init({
          env: 'your-env-id', // 替换为你的环境 ID
          traceUser: true,
        });
        console.log('云开发初始化成功');
      } catch (e) {
        console.warn('云开发初始化失败', e);
      }
    }
  }
});
```
