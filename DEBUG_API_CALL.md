# API调用失败诊断指南

## 如何查看云函数日志

### 方法1：在微信开发者工具中查看

1. **打开云开发控制台**
   - 点击工具栏的 **云开发** 按钮
   - 或菜单：**工具** → **云开发**

2. **查看云函数日志**
   - 进入 **云函数** → **imageAPI**
   - 点击 **日志** 标签
   - 查看最近的调用记录

3. **查找错误信息**
   - 查找带有 `❌` 或 `ERROR` 的日志
   - 查看完整的错误堆栈信息

### 方法2：在控制台查看

在小程序控制台中，查找：
- `❌ 调用云函数失败`
- `❌ 云函数返回错误`
- `❌ 阿里云通义万相API调用失败`

## 常见错误及解决方案

### 错误1：API Key未配置

**错误信息：**
```
❌ API Key未配置！
API Key未配置，请在云函数环境变量中配置 DASHSCOPE_API_KEY 或 ALIYUN_API_KEY
```

**解决方案：**
1. 在云开发控制台 → 云函数 → imageAPI → 配置
2. 添加环境变量：
   ```
   DASHSCOPE_API_KEY = your-api-key-here
   ```
3. 保存并重新部署云函数

### 错误2：API Key无效或过期

**错误信息：**
```
响应状态: 401
通义万相API错误: Invalid API Key
```

**解决方案：**
1. 检查API Key是否正确
2. 确认API Key是否过期
3. 在阿里云控制台重新生成API Key

### 错误3：API请求格式错误

**错误信息：**
```
响应状态: 400
通义万相API错误: Invalid request parameters
```

**解决方案：**
1. 检查请求参数格式
2. 确认模型名称是否正确：`wanx2.5-t2i-preview`
3. 确认API地址是否正确

### 错误4：网络超时

**错误信息：**
```
timeout of 120000ms exceeded
```

**解决方案：**
1. 检查网络连接
2. 图片生成可能需要较长时间，等待重试
3. 如果经常超时，可以联系阿里云技术支持

### 错误5：API配额不足

**错误信息：**
```
响应状态: 429
通义万相API错误: Rate limit exceeded
```

**解决方案：**
1. 检查API调用次数是否超限
2. 等待一段时间后重试
3. 升级API套餐

## 诊断步骤

### 步骤1：检查云函数是否部署

1. 在云开发控制台 → 云函数
2. 确认 `imageAPI` 存在且状态为"正常"
3. 如果不存在，右键 `cloudfunctions/imageAPI` → 上传并部署

### 步骤2：检查环境变量

1. 在云开发控制台 → 云函数 → imageAPI → 配置
2. 查看 **环境变量** 部分
3. 确认 `DASHSCOPE_API_KEY` 或 `ALIYUN_API_KEY` 已配置
4. 确认值是否正确：`your-api-key-here`

### 步骤3：查看云函数日志

1. 在云开发控制台 → 云函数 → imageAPI → 日志
2. 查找最近的调用记录
3. 查看是否有以下日志：
   - `🔑 API Key检查:` - 说明API Key读取情况
   - `调用阿里云通义万相2.5 API:` - 说明API调用开始
   - `API响应状态:` - 说明API响应状态
   - `❌` - 错误信息

### 步骤4：测试API Key

如果API Key可能有问题，可以：
1. 在阿里云控制台检查API Key状态
2. 使用curl或Postman测试API Key是否有效
3. 确认API Key是否有图片生成权限

## 获取详细错误信息

### 在小程序中查看

生成插画时，打开控制台，查找：
```
❌ 调用云函数失败: [错误信息]
❌ 云函数返回错误: [错误信息]
```

### 在云函数日志中查看

在云开发控制台 → 云函数 → imageAPI → 日志中，查找：
```
❌ 阿里云通义万相API调用失败: [错误信息]
响应状态: [状态码]
响应数据: [响应内容]
```

## 需要提供的信息

如果问题仍未解决，请提供：

1. **云函数日志中的完整错误信息**
   - 包括错误堆栈
   - 包括API响应数据

2. **控制台中的错误信息**
   - 包括完整的错误消息

3. **环境变量配置截图**
   - 确认环境变量名称和值

4. **API Key状态**
   - 在阿里云控制台确认API Key是否有效
