# 直接API调用配置说明

## 概述

插画生成功能已改为直接使用 `wx.request` 调用阿里云通义万相API，**不再需要云函数**。

## 重要配置步骤

### 1. 开发环境配置（必须）

在微信开发者工具中配置，否则无法调用外部API：

1. 点击右上角 **"详情"** 按钮
2. 进入 **"本地设置"** 标签页
3. **勾选**以下选项：
   - ✅ **不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**

⚠️ **注意**：此设置仅用于开发调试，正式发布前需要在微信公众平台配置域名白名单。

### 2. 正式发布前配置（必须）

如果要在正式环境使用，需要在微信公众平台配置域名白名单：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **开发** → **开发管理** → **开发设置**
3. 在 **服务器域名** 中配置：

   **request合法域名**（添加以下域名）：
   ```
   https://dashscope.aliyuncs.com
   ```

   **downloadFile合法域名**（如果API返回的是图片URL，需要添加图片下载域名）：
   ```
   https://dashscope.aliyuncs.com
   ```
   （具体域名需要根据API返回的图片URL确定）

### 3. API Key配置

API Key 不应硬编码在 `miniprogram/utils/api.js` 中。请在小程序“我的 -> API配置”中填写，或使用云函数环境变量：

```javascript
const apiKey = 'your-api-key-here'
```

⚠️ **安全提示**：
- 不要将真实 API Key 提交到 Git
- 前端本地配置只适合个人开发测试
- 正式环境建议使用云函数环境变量保护 API Key

### 4. 测试步骤

1. 确保已勾选"不校验合法域名"（开发环境）
2. 打开小程序，进入插画生成页面
3. 输入描述并生成插画
4. 查看控制台日志，确认API调用成功

## 常见问题

### Q1: 提示"网络请求失败"或"域名白名单"错误

**解决方案**：
- 检查是否已勾选"不校验合法域名"（开发环境）
- 检查网络连接是否正常
- 正式环境需要配置域名白名单

### Q2: 提示"API调用被拒绝(403)"

**可能原因**：
- API Key无效或过期
- 未开通通义万相服务
- API Key没有图片生成权限

**解决方案**：
- 检查API Key是否正确
- 确认已开通通义万相服务
- 确认API Key有图片生成权限

### Q3: 图片下载失败

**可能原因**：
- 图片URL域名未配置白名单
- 网络连接问题

**解决方案**：
- 配置downloadFile合法域名
- 检查网络连接

## 代码位置

- API调用函数：`miniprogram/utils/api.js` → `callImageAPI()`
- API Key配置：`miniprogram/utils/api.js` 第703行

## 优势

✅ **无需部署云函数**：直接在前端调用，简化部署流程  
✅ **减少延迟**：跳过云函数中转，直接调用API  
✅ **简化调试**：可以直接在前端控制台查看请求和响应

## 劣势

⚠️ **安全风险**：API Key暴露在前端代码中  
⚠️ **需要配置域名白名单**：正式发布前必须配置  
⚠️ **无法隐藏API Key**：任何人都可以查看前端代码获取API Key

## 建议

如果API Key需要保密，建议：
1. 开发测试：使用当前直接调用方式
2. 正式发布：改回使用云函数，保护API Key
