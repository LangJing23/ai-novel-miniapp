# 图片生成API配置完整指南

## 重要说明

⚠️ **混元、DeepSeek-V3、Aliyun 都是文本生成模型，不能直接生成图片！**

图片生成需要使用专门的**图片生成API**，例如：
- 文心一格（百度）
- 通义万相（阿里云）
- Stable Diffusion
- DALL-E

## 当前支持的图片生成API

### 1. 阿里云通义万相（优先）

**优势：**
- 阿里云官方服务，稳定可靠
- 支持中文提示词
- 图片质量高

**配置步骤：**

1. **获取API密钥**
   - 访问 [阿里云控制台](https://ecs.console.aliyun.com/)
   - 进入 **模型服务灵积** 或 **通义万相**
   - 创建API Key，获取：
     - `API Key`
     - `API Secret`

2. **配置环境变量**
   - 在云开发控制台 → 云函数 → imageAPI → 配置
   - 添加环境变量：
     ```
     ALIYUN_API_KEY = 你的API Key
     ALIYUN_API_SECRET = 你的API Secret
     ```

3. **API文档**
   - 官方文档：https://help.aliyun.com/zh/model-studio/developer-reference/api-details-9

### 2. 文心一格（备用）

**配置步骤：**

1. **获取API密钥**
   - 访问 [百度智能云控制台](https://console.bce.baidu.com/)
   - 进入 **千帆大模型平台**
   - 创建应用，获取：
     - `API Key`
     - `Secret Key`

2. **配置环境变量**
   ```
   WENXIN_API_KEY = 你的API Key
   WENXIN_SECRET_KEY = 你的Secret Key
   ```

## API调用优先级

代码会按以下顺序尝试：

1. **阿里云通义万相**（如果配置了ALIYUN_API_KEY）
2. **文心一格**（如果配置了WENXIN_API_KEY）
3. **模拟模式**（如果都没有配置）

## 检查当前配置

### 方法1：查看云函数日志

1. 打开云开发控制台
2. 进入 **云函数** → **imageAPI** → **日志**
3. 查看最近的调用日志：
   - 如果看到 `📞 尝试使用阿里云通义万相...`，说明正在尝试阿里云
   - 如果看到 `📞 尝试使用文心一格...`，说明正在尝试文心一格
   - 如果看到 `⚠️ 所有API都失败，使用模拟模式`，说明没有配置任何API

### 方法2：查看环境变量

1. 在云开发控制台 → 云函数 → imageAPI → 配置
2. 查看 **环境变量** 部分
3. 确认是否配置了：
   - `ALIYUN_API_KEY` 和 `ALIYUN_API_SECRET`（阿里云）
   - 或 `WENXIN_API_KEY` 和 `WENXIN_SECRET_KEY`（文心一格）

## 常见问题

### Q1: 为什么不能使用混元/DeepSeek生成图片？

**A:** 混元和DeepSeek是**文本生成模型**，只能生成文字，不能生成图片。图片生成需要使用专门的**图片生成模型**，如通义万相、文心一格等。

### Q2: 如何选择图片生成API？

**推荐顺序：**
1. **阿里云通义万相** - 如果已有阿里云账号
2. **文心一格** - 如果已有百度智能云账号
3. 其他第三方API（需要自行集成）

### Q3: API调用失败怎么办？

**检查清单：**
- [ ] API密钥是否正确配置
- [ ] 环境变量名称是否正确
- [ ] API密钥是否有效（未过期）
- [ ] 网络连接是否正常
- [ ] 查看云函数日志获取详细错误信息

### Q4: 如何验证API是否配置成功？

1. 在小程序中生成插画
2. 查看控制台日志：
   - 如果看到 `✅ 阿里云通义万相调用成功` 或 `✅ 文心一格调用成功`
   - 并且返回的不是透明占位图
   - 说明配置成功
